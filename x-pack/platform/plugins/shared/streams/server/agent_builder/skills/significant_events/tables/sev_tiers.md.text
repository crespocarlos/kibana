## Criticality tiers

| Range | Tier | Description | Response |
|---|---|---|---|
| 76–100 | SEV1 | Core user journey completely broken, or PII/credentials confirmed in log output. Broad or global impact. | Page immediately. |
| 51–75 | SEV2 | Significant feature unavailable, meaningful customer pain, no viable workaround. | Respond within the hour. |
| 31–50 | SEV3 | Partial degradation, limited blast radius, stable workarounds exist. | Schedule a fix. |
| 0–30 | SEV4/5 | Low-impact issue, noise, or confirmed false alarm. | Demote or monitor. |

**Override cap:**
- When uncertain between tiers, choose the lower one.
